# Pricing Plan Layout Page

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to build a responsive pricing comparison layout with multiple plan cards, feature lists, icons, and clear call-to-action buttons.

## Features
- Three pricing cards arranged in a responsive flexbox layout
- Distinct plan tiers for Basic, Pro, and Premium options
- Feature lists with active and disabled states
- Icon styling using Bootstrap Icons
- Clear visual hierarchy with headings, prices, and action buttons
- Consistent card spacing, borders, and rounded corners

## Technologies Used
- HTML5
- CSS3
- Bootstrap Icons

## What I Learned
- How to structure multi-card pricing layouts with Flexbox
- Styling feature lists with active and disabled states
- Using icons to improve scanability and visual clarity
- Building consistent card components with shared layout patterns
- Creating a responsive interface that keeps pricing plans easy to compare

## Sections Included
- Page heading
- Three pricing plan cards
- Feature lists for each plan
- Get Started buttons for each tier

## CSS Concepts Practiced
- Flexbox layout and spacing
- Card styling with borders, shadows, and rounded corners
- Typography hierarchy for pricing content
- Icon alignment within list items
- Disabled state styling for unavailable features
- Responsive container sizing
