# Build a Cat Photo App

This project is part of the freeCodeCamp Responsive Web Design certification.

The Cat Photo App introduces fundamental HTML concepts by building a simple webpage with images, links, lists, and semantic elements.

## Features
- Semantic HTML5 structure
- External links with anchor elements
- Responsive images with descriptive alt text
- Ordered and unordered lists
- Figure and figcaption elements
- Text formatting with `em` and `strong`

## Technologies Used
- HTML5

## What I Learned
- How to structure a webpage using semantic HTML
- How to add images and hyperlinks
- The importance of accessibility using `alt` attributes
- How to create ordered and unordered lists
- How to group media content using `figure` and `figcaption`

## Sections Included
- Cat Photos
- Things Cats Love
- Top 3 Things Cats Hate
