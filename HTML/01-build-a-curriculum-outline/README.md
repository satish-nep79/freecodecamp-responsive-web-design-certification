# Build a Curriculum Outline

This project is part of the freeCodeCamp Responsive Web Design certification.

It introduces the basic structure of an HTML webpage using headings and paragraphs to create a simple curriculum outline.

## Features
- Basic HTML5 document structure
- Use of semantic heading elements (`h1`–`h3`)
- Paragraph content for descriptions
- Responsive meta viewport setup

## Technologies Used
- HTML5

## What I Learned
- How to structure a webpage with headings
- How to organize content hierarchically
- How to create a valid HTML document
