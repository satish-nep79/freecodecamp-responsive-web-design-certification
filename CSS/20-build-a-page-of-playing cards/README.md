# Build a Page of Playing Cards

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to generate a deck-style layout of playing cards from JSON data and render each card with suit-specific styling, face-card treatment, and a clean responsive arrangement.

## Features
- Cards rendered dynamically from a JSON data file
- Suit-based styling for red and black cards
- Special face-card layouts for J, Q, and K cards
- Symbol placement on both corners of each card
- Responsive card grid with consistent spacing
- Error handling for failed data loading

## Technologies Used
- HTML5
- CSS3
- JavaScript

## What I Learned
- How to build UI from external JSON data
- Rendering repeated card components with JavaScript loops and template literals
- Applying conditional classes based on card metadata
- Structuring card faces with nested layout elements
- Handling fetch errors gracefully in the browser

## Sections Included
- Main card container
- Rendered playing cards
- Face-card symbol layouts
- Error message fallback state

## CSS Concepts Practiced
- Card styling and spacing
- Color-based class switching
- Layout composition with nested elements
- Responsive container sizing
- Typography and symbol alignment
