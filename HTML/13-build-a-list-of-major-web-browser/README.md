# Build a List of Major Web Browsers

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create a structured webpage displaying major web browsers and their descriptions using HTML description lists.

## Features
- Semantic HTML5 structure
- Description list using `dl`, `dt`, and `dd`
- Informative browser descriptions
- Organized and readable content layout
- Responsive webpage setup

## Technologies Used
- HTML5

## What I Learned
- How to use description lists in HTML
- The purpose of `dl`, `dt`, and `dd` elements
- How to organize informational content semantically
- The importance of clean document structure
- How semantic HTML improves readability and accessibility

## Browsers Included
- Google Chrome
- Firefox
- Safari
- Brave
- Arc

## HTML Concepts Practiced
- Description list elements (dl, dt, dd)
- Semantic markup for definitions
- Content organization with description lists
- Heading hierarchy
- Structured information presentation
- Accessibility with semantic lists
