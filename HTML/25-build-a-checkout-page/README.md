# Build a Checkout Page

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to build a structured checkout page using semantic HTML and accessible form practices, including required fields and ARIA attributes.

## Features
- Semantic HTML layout using sections
- Cart section displaying an item image
- Payment form with labeled inputs
- Required fields marked with `*`
- Accessibility improvements using `aria-describedby`
- Help text for card number formatting
- Proper association between labels and inputs

## Technologies Used
- HTML5

## What I Learned
- How to structure a checkout flow using semantic HTML
- How to improve form accessibility using ARIA attributes
- How to provide inline validation guidance with help text
- How to visually indicate required fields in forms
- How to design user-friendly payment forms

## Sections Included
- Cart Summary
- Payment Information Form

## Form Fields Included
- Cardholder Name
- Card Number (with format guidance)
