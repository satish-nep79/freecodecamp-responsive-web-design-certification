# Build a Travel Agency Page

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create a travel agency webpage featuring travel packages, destination itineraries, and linked images using semantic HTML.

## Features
- Semantic HTML5 document structure
- SEO meta description
- Travel package listings with external links
- Destination itinerary showcase using figures and captions
- Images with descriptive alt text for accessibility
- External links opening in new tabs

## Technologies Used
- HTML5

## What I Learned
- How to structure a complete webpage using semantic HTML
- How to improve accessibility with meaningful alt text
- How to organize media content with `figure` and `figcaption`
- How to create SEO-friendly meta descriptions
- How to use external links with the `target="_blank"` attribute

## Sections Included
- Travel Packages
- Group Travels
- Private Tours
- Top Destination Itineraries
