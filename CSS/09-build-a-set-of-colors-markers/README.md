# Build a Set of Color Markers

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to explore color theory and practical color usage in CSS by building a simple set of marker graphics using gradients, color spaces (RGB, HSL), and shadows.

## Features
- Marker-style graphics composed of caps and sleeves
- Linear gradients using RGB and HSL color notations
- Subtle box-shadows for depth and glow effects
- Demonstrations of contrast and layering with semi-transparent fills

## Technologies Used
- HTML5
- CSS3

## What I Learned
- Practical use of linear gradients to create depth and highlights
- When to prefer HSL vs RGB for hue/saturation/lightness control
- How to add atmosphere with box-shadows and semi-transparent overlays
- Importance of contrast and color harmony for legibility and visual impact

## Sections Included
- Red marker example
- Green marker example
- Blue marker example

## CSS Concepts Practiced
- Linear gradients and multiple color stops
- HSL and RGB color models
- Box-shadow and layering
- Semi-transparent colors with rgba/hsla
