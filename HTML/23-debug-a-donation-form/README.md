# Debug a Donation Form

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to fix syntax and accessibility issues in a donation form, ensuring proper labels, input types, and required validation for a better user experience.

## Features
- Semantic HTML form structure
- Properly associated labels using `for` and `id`
- Correct input types for validation (`text`, `email`, `number`)
- Required fields for essential user input
- Checkbox option for newsletter subscription
- Submit button for form submission
- Cleaned and corrected HTML syntax

## Technologies Used
- HTML5

## What I Learned
- How to properly associate labels with form inputs for accessibility
- How to use correct input types for validation
- How to improve form usability with `required` attributes
- How to debug common HTML form mistakes
- How small fixes can significantly improve accessibility

## Form Includes
- Full Name field
- Email Address field
- Donation Amount field
- Newsletter subscription checkbox
