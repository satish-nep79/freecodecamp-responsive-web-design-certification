# Design a Set of Colorful Boxes

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create a responsive grid of colorful boxes using HTML and CSS Flexbox, demonstrating flexible layout, color styling, and advanced flexbox properties like `flex-grow`, `flex-shrink`, and `order`.

## Features
- Flexbox container with wrapping layout
- Six colorful boxes with distinct background colors
- Dynamic box sizing with `flex-grow` and `flex-shrink` properties
- Custom ordering using the `order` property
- Responsive container with center alignment
- Styled headings and typography within boxes
- Border-radius for rounded corners
- Max-height constraints and consistent spacing

## Technologies Used
- HTML5
- CSS3

## What I Learned
- How to structure flexible box layouts with Flexbox
- Using `flex-grow` to allow boxes to expand
- Using `flex-shrink` to control box shrinking behavior
- Implementing custom order with the `order` property
- Creating responsive container sizing
- Styling and organizing multiple colored elements
- Using flexbox alignment properties (`align-items`, `align-content`)
- Working with flex basis and flex direction

## Sections Included
- Header with title
- Six colored boxes in flexbox container
- Color labels for each box (Red, Orange, Yellow, Green, Blue, Indigo)

## CSS Concepts Practiced
- Flexbox container (`display: flex`)
- Flex properties (`flex`, `flex-grow`, `flex-shrink`, `flex-basis`)
- Flex direction and wrapping
- Flex alignment (`align-items`, `align-content`)
- Box sizing and max-height constraints
- Border-radius for rounded corners
- Custom box ordering with `order` property
- Color styling and backgrounds
- Typography and text alignment
- Margin and padding spacing
