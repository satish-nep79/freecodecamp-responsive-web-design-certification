# Build a Game Settings Panel

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to build a game settings panel UI using HTML and CSS, focusing on custom checkbox design, card layout, and clean interaction styling.

## Features
- Centered settings card layout with a compact design
- Custom checkbox styling for selected and unselected states
- Smooth transition effects for checkbox interaction
- Clean typography and spacing for game setting options
- Visual focus on accessible game UI controls

## Technologies Used
- HTML5
- CSS3

## What I Learned
- How to build custom checkbox controls with CSS only
- How to use `appearance: none` to override browser default form controls
- How to style checked states and add a custom checkmark
- How to design a compact, card-style UI for settings panels
- How to space elements clearly and create a readable label layout

## Sections Included
- Sound Effects toggle
- Background Music toggle
- Hard Mode toggle
- Haptic Feedback toggle

## CSS Concepts Practiced
- Custom form control styling
- Checkbox `:checked` state styling
- `appearance: none` and browser control normalization
- Box-shadow and border-radius for panel design
- Layout with centered cards and vertical spacing
