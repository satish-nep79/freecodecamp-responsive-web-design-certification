# Design a Cafe Menu

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to design and style a responsive cafe menu using HTML and CSS, focusing on layout design, typography, spacing, and visual styling.

## Features
- Responsive cafe menu layout
- Styled menu sections for coffee and desserts
- Custom typography and spacing
- Background image styling
- Interactive link hover effects
- Organized pricing layout using inline-block elements
- Semantic HTML structure with accessible images

## Technologies Used
- HTML5
- CSS3

## What I Learned
- How to style webpages using CSS
- How to center layouts using auto margins
- How to use background images effectively
- How to control spacing with margin and padding
- How to align content using inline-block elements
- How to style links with pseudo-classes like `:hover` and `:active`
- How to build responsive layouts using `max-width`

## Sections Included
- Coffee Menu
- Dessert Menu
- Footer with Contact Information

## CSS Concepts Practiced
- Selectors and grouping
- Typography styling
- Colors and backgrounds
- Layout alignment
- Pseudo-classes
- Responsive sizing
