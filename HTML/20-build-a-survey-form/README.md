# Build a Survey Form

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to build a fully functional survey form as part of the certification projects, using a variety of HTML form elements and validation rules.

## Features
- Semantic HTML form structure
- Required input fields for name, email, and age
- HTML5 validation for email and number inputs
- Dropdown selection using `select` and `option`
- Radio button group for single-choice answers
- Checkbox group for multiple selections
- Textarea for open-ended feedback
- Submit button with form submission support

## Technologies Used
- HTML5
- CSS (linked stylesheet)

## What I Learned
- How to build structured and accessible forms
- How to use form validation attributes like `required`, `min`, and `max`
- How to group inputs using `fieldset`
- How to work with different input types effectively
- How to design user-friendly survey interfaces

## Sections Included
- Personal Information
- Feature Selection
- Recommendation Choice
- Improvement Feedback
- Additional Comments
