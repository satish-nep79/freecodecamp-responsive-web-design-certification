# Build a Video Display Using an iframe

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to embed and display a video on a webpage using the HTML `iframe` element.

## Features
- Embedded YouTube video using an `iframe`
- Responsive video dimensions
- Fullscreen video support
- Secure embedding attributes for improved browser compatibility
- Simple and clean page structure

## Technologies Used
- HTML5

## What I Learned
- How to use the HTML `iframe` element
- How to embed external video content into a webpage
- The purpose of iframe attributes like `allowfullscreen`
- How embedded media improves user experience
- The importance of secure embedding policies

## Media Included
- Embedded YouTube video display

## HTML Concepts Practiced
- HTML5 iframe element
- Embedded external content
- iframe attributes (allowfullscreen, src)
- Security considerations for iframes
- Responsive iframe embedding
- Browser compatibility for embedded media
