# Build a Flexbox Photo Gallery

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create and style a responsive photo gallery webpage using HTML, CSS, and JavaScript, with a focus on Flexbox layout, responsive image handling, dynamic content rendering, and elegant card-based design.

## Features
- Responsive flexbox-based photo gallery layout
- Dynamic image rendering using JavaScript ES6 templates
- Elegant card-style gallery items with rounded corners
- Smooth hover animations with scale and shadow transitions
- Professional metadata display with image titles and descriptions
- CSS custom properties for consistent theming and easy customization
- Soft, studio-inspired color palette
- Header with gallery title and description section
- Figure and figcaption semantic HTML for accessibility

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## What I Learned
- How to structure reusable photo gallery layouts with Flexbox
- Dynamic content rendering using JavaScript template literals and loops
- Creating responsive card-based components
- Styling images with `object-fit` for consistent aspect ratios
- Using CSS custom properties (variables) for theme management
- Implementing smooth hover animations with transitions and transforms
- Building accessible image galleries with proper semantic markup
- Designing modern, minimalist interfaces with subtle shadows and spacing
- Organizing layout with Flexbox for flexible, responsive arrangements

## Sections Included
- Header with gallery title
- Gallery heading with description
- Photo gallery grid with dynamic image cards
- Image metadata with titles and descriptions

## CSS Concepts Practiced
- Flexbox layout (`display: flex`, `flex-wrap`, `justify-content`, `align-items`, `gap`)
- CSS custom properties (variables) for theming
- Responsive sizing with `max-width` and `fit-content`
- Object-fit for image handling
- Hover effects with transitions and transform scaling
- Box shadows for depth and elevation
- Border-radius for subtle modern corners
- Typography styling with custom font families
- Box-sizing for consistent spacing calculations
- Overflow handling for image containment
- Background color theming

## Styling Highlights
- Soft studio off-white background (#fafafa)
- Pure white card containers with subtle shadows
- High contrast typography for readability
- Smooth cubic-bezier animations (0.25, 1, 0.5, 1)
- Subtle hover scale transformation (1.05)
- Fine structural borders and spacing
- Professional color palette with main, muted text, and border colors
