# FreeCodeCamp Responsive Web Design Projects

This repository contains completed projects for the freeCodeCamp Responsive Web Design certification. The work is organized into two main sections:

- `HTML/` for semantic HTML, multimedia embedding, forms, tables, accessibility, and debugging exercises.
- `CSS/` for responsive layout, visual styling, card design, hover effects, gradients, and modern CSS techniques.

## Section Summaries

### HTML Projects
The HTML section includes a wide range of practice projects covering HTML structure and semantics, multimedia content, embedded video and audio, blog layouts, forms, tables, and accessibility improvements.

### CSS Projects
The CSS section contains styled interface projects such as a cafe menu, business card, stylized to-do list, blog post card, and event flyer page. These projects demonstrate responsive card layouts, hover states, transitions, and decorative visuals.

## Live Preview
You can browse all live previews and project details directly from the hosted portfolio:

https://satish-nep79.github.io/freecodecamp-responsive-web-design-certification/

## Project Files

- `index.html` is the main portfolio page that loads project cards and previews.
- `style.css` contains the root page styles for the portfolio interface.
- `projects.js` stores the project metadata used to render each project card.

## Usage

Open `index.html` in a browser to view the project gallery and access live previews and notes for each project.
