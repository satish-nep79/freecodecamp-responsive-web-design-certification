# Build an Event Flyer Page

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to design and style an event flyer webpage using semantic HTML and modern CSS techniques. The project focused on layout structure, spacing, responsive sizing, decorative styling, and creating an engaging event presentation page.

## Features
- Hero banner with featured event image
- Custom event flyer layout
- Multiple event information sections
- Decorative horizontal dividers
- Responsive centered page design
- Styled event timeline and feature lists
- Custom patterned background using CSS gradients
- Themed color palette inspired by the Tihar festival

## Technologies Used
- HTML5
- CSS3

## What I Learned
- How to structure webpages using semantic HTML elements
- Creating responsive layouts using viewport-based sizing
- Using the `calc()` function for dynamic sizing
- Styling sections and containers with Flexbox
- Designing decorative backgrounds with layered CSS gradients
- Positioning elements using relative and absolute positioning
- Creating visually balanced flyer layouts with spacing and alignment
- Building reusable color systems with CSS custom properties (variables)
