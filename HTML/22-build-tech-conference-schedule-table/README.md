# Build a Tech Conference Schedule Table

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create a structured and accessible conference schedule using an HTML table with multiple tracks, time slots, and grouped sessions.

## Features
- Semantic HTML table structure
- Table caption for context
- Column headers using `thead`
- Structured schedule using `tbody`
- Row headers for time slots using `scope="row"`
- Multi-column layout using `colspan`
- Organized conference agenda format

## Technologies Used
- HTML5

## What I Learned
- How to build complex tables with multiple sections
- How to use `scope` for better accessibility
- How to structure schedules using tabular data
- How to merge cells using `colspan`
- How to present structured information clearly
