# Build an HTML Video Player

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create a simple video player webpage using the HTML video element with multiple video formats and playback features.

## Features
- Semantic HTML5 structure
- Embedded video player using the `video` element
- Playback controls for user interaction
- Loop and muted functionality
- Poster image displayed before playback
- Multiple video source formats for browser compatibility

## Technologies Used
- HTML5

## What I Learned
- How to use the HTML `video` element
- How to provide multiple video formats using `source`
- How to add playback controls and looping functionality
- The importance of browser compatibility for media files
- How to use poster images for video previews

## Media Formats Included
- MP4
- WebM
- OGG
- MOV
