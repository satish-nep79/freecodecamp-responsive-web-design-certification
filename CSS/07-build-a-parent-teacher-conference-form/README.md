# Design a Parent-Teacher Conference Form

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create and style a parent-teacher conference form using HTML and CSS, with a focus on layout, form structure, accessible input styling, and custom radio controls.

## Features
- Centered conference form layout with card-style container
- Clear fieldset grouping for student, parent, and contact sections
- Styled text inputs and textarea with transparent background
- Custom radio button styling for contact method selection
- Accessible focus and hover states for interactive elements
- Submit button with hover transition
- Dark-themed interface with contrast styling

## Technologies Used
- HTML5
- CSS3

## What I Learned
- How to style form layouts using fieldsets and legends
- How to build custom radio controls with CSS pseudo-elements
- How to style inputs and textarea with transparent backgrounds
- How to use box shadows, border-radius, and centered containers
- How to create accessible and readable form sections
- How to design a dark UI with legible contrast

## Sections Included
- Student Information
- Parent/Guardian Information
- Preferred Contact Method
- Additional Notes

## CSS Concepts Practiced
- Form styling and layout
- Fieldset and legend styling
- Custom radio buttons with `::before`
- Input and textarea styling
- Button hover effects
- Background and text contrast
- Container centering and responsive widths
