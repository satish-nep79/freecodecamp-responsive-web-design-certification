# Build a Nutrition Label

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to build a nutrition-label style product card layout from JSON data, combining structured content, bold typography, and clear information hierarchy.

## Features
- Nutrition-style label layout built for each product card
- Product data loaded from a JSON file
- Structured sections for serving information, calories, and daily values
- Repeated nutrient rows for vitamins and minerals
- Responsive card layout with product image and nutrition panel side by side
- Error message fallback when data loading fails

## Technologies Used
- HTML5
- CSS3
- JavaScript

## What I Learned
- How to render repeated UI from fetched JSON data
- Structuring content-heavy layouts with nested containers
- Creating a nutrition facts visual hierarchy with spacing and borders
- Using typography to emphasize labels, totals, and supporting text
- Handling asynchronous loading and error states in the browser

## Sections Included
- Product details block
- Nutrition facts label
- Daily value breakdown
- Vitamins and minerals list
- Error fallback message

## CSS Concepts Practiced
- Flexbox card layout
- Border and divider styling
- Typography hierarchy and alignment
- Spacing and grouping for data-heavy interfaces
- Responsive image sizing
