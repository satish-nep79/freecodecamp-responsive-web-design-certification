# Build a Cat Blog Page

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create a structured blog webpage featuring articles, navigation links, images, and contact information using semantic HTML elements.

## Features
- Semantic HTML5 page structure
- Navigation menu with internal anchor links
- Blog articles using the `article` element
- Featured image with `figure` and `figcaption`
- Footer section with contact information
- Accessible phone and email links

## Technologies Used
- HTML5

## What I Learned
- How to structure a blog layout using semantic HTML
- How to create internal navigation with anchor links
- How to organize content using `section` and `article`
- How to display media with `figure` and `figcaption`
- How to use the `address` element for contact information

## Sections Included
- About
- Blog Posts
- Contact Information

## Blog Posts Included
- Mr. Whiskers' First Day Home
- Mr. Whiskers' First Bath
- Mr. Whiskers' First Birthday Party

## HTML Concepts Practiced
- Semantic blog layout with header, nav, main, article, footer
- Internal anchor navigation for page sections
- Article elements for content grouping
- Figure and figcaption for media presentation
- Address element for contact information
- Heading hierarchy for content organization
- Accessible navigation and linking
