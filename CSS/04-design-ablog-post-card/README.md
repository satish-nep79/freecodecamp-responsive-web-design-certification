# Design a Blog Post Card

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create and style a responsive blog post card using HTML and CSS. The project focused on practicing backgrounds, borders, spacing, hover effects, and card layouts while building a clean and modern blog-style interface.

## Features
- Responsive blog post card layout
- Dynamic rendering of blog cards using JavaScript
- Styled card design with rounded corners and hover animations
- Responsive image handling with full-width card images
- Custom “Read More” button with hover effects
- Flexbox layout for arranging multiple blog cards
- External links opening in a new browser tab

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

## What I Learned
- How to structure reusable card components
- Styling layouts using Flexbox
- Creating smooth hover animations with transitions and transforms
- Working with dynamic content using JavaScript arrays and template literals
- Designing responsive card-based UI layouts
- Improving visual hierarchy using spacing, typography, and colors
