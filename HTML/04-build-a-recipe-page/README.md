# Build a Recipe Page

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create a simple recipe webpage using basic HTML elements and semantic structure.

## Features
- Semantic HTML5 document structure
- Recipe title and description
- Ingredients section with an unordered list
- Step-by-step instructions using an ordered list
- Recipe image with descriptive alt text

## Technologies Used
- HTML5

## What I Learned
- How to structure content with headings and paragraphs
- How to organize data using lists
- The importance of semantic HTML elements
- How to add images and improve accessibility with `alt` attributes
- How to build a complete webpage from scratch

## Recipe Included
- Easy Banana Pancakes
