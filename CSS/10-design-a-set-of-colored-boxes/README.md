# Design a Set of Colored Boxes

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to explore color representation and UI presentation by building a grid of colored cards using multiple color formats and CSS techniques.

## Features
- Grid of colored boxes showcasing HEX, RGB, named color, and HSL formats
- Smooth hover interactions with scale and shadow
- Background switching via `:has()` hover selectors on the page
- Frosted glass content panels using semi-transparent backgrounds and backdrop-filter
- Consistent theming via CSS custom properties

## Technologies Used
- HTML5
- CSS3

## What I Learned
- How to use CSS custom properties to centralize color values
- Differences between color formats (HEX, RGB, HSL, named colors)
- How to use `:has()` to change page background based on hovered element
- Creating accessible contrast and readable content panels over colorful backgrounds
- Using transitions, transforms, shadows, and backdrop filters for visual polish

## Sections Included
- Color examples (HEX, RGB, named color, HSL, custom combo)
- Hover interactions that lift and highlight each box

## CSS Concepts Practiced
- CSS variables and theming
- `:has()` and modern selector usage
- Transforms and transitions for micro-interactions
- Backdrop-filter and semi-transparent layering
- Box-shadow, borders, and text-shadow for legibility
