# Build a Book Catalog Table

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create a structured book catalog using an HTML table, organizing book information into a clear and readable format.

## Features
- Semantic HTML table structure
- Table header with column titles (Title, Author, Genre, Publication Year)
- Table body with multiple book entries
- Table footer with total count of books
- Use of `colspan` to span footer text across columns
- Organized tabular layout for structured data

## Technologies Used
- HTML5

## What I Learned
- How to structure complex tables in HTML
- How to separate table content into `thead`, `tbody`, and `tfoot`
- How to display structured datasets clearly in a webpage
- How to use `colspan` for summary rows
- How to build catalog-style layouts using tables
