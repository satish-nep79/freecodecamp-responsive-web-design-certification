# Build a Multimedia Player

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create a multimedia player webpage featuring audio playback, video playback with subtitles, and a transcript section using semantic HTML and accessible media elements.

## Features
- Semantic HTML5 page structure
- Audio player with accessible labels
- Video player with playback controls
- Subtitle support using the `track` element
- Transcript section for accessibility
- Responsive media layout with defined video width

## Technologies Used
- HTML5

## What I Learned
- How to embed audio and video content in HTML
- How to improve accessibility using `aria-label`
- How to add subtitles using the `track` element
- How to provide transcripts for multimedia content
- How to structure multimedia-focused webpages semantically

## Sections Included
- Audio Player
- Video Player
- Video Transcript

## Media Included
- Sailing Away audio track
- JavaScript map method educational video
- English subtitle track
