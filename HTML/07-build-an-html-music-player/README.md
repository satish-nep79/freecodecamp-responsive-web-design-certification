# Build an HTML Music Player

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create a simple music player webpage using the HTML audio element to play multiple songs directly in the browser.

## Features
- Semantic HTML5 structure
- Embedded audio players using the `audio` element
- Playback controls for each track
- Loop functionality for continuous playback
- Multiple song sections with artist information

## Technologies Used
- HTML5

## What I Learned
- How to use the HTML `audio` element
- How to add playback controls to audio files
- How to create looping audio tracks
- How to organize multimedia content on a webpage
- The importance of semantic structure for readability

## Songs Included
- Can't Stay Down
- Cruising for a Musing
- Scratching the Surface

## HTML Concepts Practiced
- HTML5 audio element
- Playback controls and attributes
- Audio source formats
- Section elements for content organization
- Semantic structure for multimedia content
- Accessible audio player implementation
