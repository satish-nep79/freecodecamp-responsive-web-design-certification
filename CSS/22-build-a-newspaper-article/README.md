# Build a Newspaper Article

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create a newspaper-style article layout with a strong typographic hierarchy, featured image, and readable editorial content flow.

## Features
- Newspaper-inspired headline and subheadline structure
- Large featured image with author and date information
- Multi-paragraph article content with justified text
- Drop-cap styling for the opening paragraph
- Clean spacing and strong visual hierarchy

## Technologies Used
- HTML5
- CSS3

## What I Learned
- How to build editorial layouts with typography as the main design tool
- Structuring article content with semantic HTML elements
- Using flexbox for aligning image and story content
- Creating a readable page with spacing, contrast, and hierarchy
- Styling a drop cap with first-letter pseudo-element

## Sections Included
- Article header
- Featured image and publication details
- Main article content
- Supporting body text block

## CSS Concepts Practiced
- Typography sizing and hierarchy
- Flexbox layout and spacing
- Text justification and line-height control
- First-letter styling
- Color contrast and muted supporting text
