# Debug Camperbot's Profile Page

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this exercise was to identify and fix invalid HTML elements and syntax errors in a simple profile page.

## Features
- Correct use of heading elements (`h1`, `h2`, `h3`)
- Proper paragraph elements (`p`)
- Fixed HTML syntax errors
- Clean and readable page structure

## Technologies Used
- HTML5

## What I Learned
- How to debug incorrect HTML tags
- The importance of valid semantic HTML
- How heading hierarchy improves structure and readability
- How to properly close HTML elements

## Final Output
The corrected profile page includes:
- A main heading
- An About section
- Two paragraphs
- A Background and Interests subheading
