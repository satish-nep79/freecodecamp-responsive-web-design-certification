# Build a Stylized To-Do List

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create and style an interactive to-do list webpage using HTML and CSS, focusing on link states, accessibility, nested lists, and modern UI styling.

## Features
- Interactive to-do checklist
- Nested sub-item lists with external product links
- Custom hover, focus, visited, and active link styles
- Accessible keyboard focus indicators
- Modern dark-themed interface
- Responsive centered layout
- Smooth transition effects for user interaction

## Technologies Used
- HTML5
- CSS3

## What I Learned
- How to style links using CSS pseudo-classes
- How to create accessible focus states using outlines
- How to build nested list structures
- How to design modern dark-themed layouts
- How to use transitions for smoother UI interactions
- How to organize content using containers and spacing

## Sections Included
- To-Do Task List
- Product Recommendation Links
- Nested Sub-Items
- Interactive Link States

## CSS Concepts Practiced
- Pseudo-classes (`:hover`, `:focus`, `:visited`, `:active`)
- Dark UI styling
- Borders and border-radius
- Box shadows
- Typography styling
- Spacing and layout
- Transitions and animations
- Accessibility styling
