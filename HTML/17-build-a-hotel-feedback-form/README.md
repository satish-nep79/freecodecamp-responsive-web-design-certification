# Build a Hotel Feedback Form

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create a structured hotel feedback form using semantic HTML form elements, allowing users to submit their experience and ratings.

## Features
- Semantic HTML5 page structure
- Complete feedback form using `<form>`
- Input validation with required fields
- Radio buttons for single-choice questions
- Checkboxes for multiple-choice selections
- Dropdown menus for ratings
- Textarea for open-ended feedback
- Organized grouping using `fieldset` and `legend`

## Technologies Used
- HTML5

## What I Learned
- How to build structured forms using HTML
- How to use different input types effectively
- How to group related form fields with `fieldset`
- How to improve usability with labels and placeholders
- How to collect structured user feedback

## Form Sections Included
- Personal Information
- First-time Stay Question
- Stay Motivation
- Service and Food Ratings
- Additional Comments
