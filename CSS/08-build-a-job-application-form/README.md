# Design a Job Application Form

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to build and style a job application form using semantic HTML and modern CSS techniques, focusing on form layout, validation UI, and custom form controls.

## Features
- Centered, card-style form container with readable width
- Required field indicators and built-in validation styling
- Position select dropdown and cover letter textarea
- Availability radio group with custom-styled radio controls
- Accessible focus and hover states for inputs and buttons
- Responsive form layout using CSS variables for consistent sizing
- Subtle shadows and micro-interactions for improved affordance

## Technologies Used
- HTML5
- CSS3

## What I Learned
- How to structure and style forms for clarity and usability
- Using CSS custom properties for theme and spacing control
- Implementing custom radio buttons and interactive focus states
- Styling valid/invalid inputs and adding visual validation cues
- Applying box shadows, rounded corners, and smooth transitions
- Using modern selectors to build concise form rules

## Sections Included
- Personal information fields (name, email)
- Position selection (dropdown)
- Availability radio group
- Cover letter / message textarea
- Submit action

## CSS Concepts Practiced
- CSS custom properties (variables)
- Form layout and input styling
- Custom radio controls and :checked states
- Validation styling (`:valid` / `:invalid`)
- Transitions, shadows, and micro-interactions
- Responsive container sizing and readable line lengths
