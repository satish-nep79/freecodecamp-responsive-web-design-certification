# Design a Greeting Card

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create and style an interactive greeting card webpage using HTML and CSS, focusing on hover effects, transitions, pseudo-classes, and target-based section switching.

## Features
- Centered greeting card design with dark theme
- Smooth hover animations with scale and background transitions
- Interactive card links with hover and active states
- Target-based section switching without JavaScript
- Emoji decorations using CSS pseudo-elements
- Box shadow and border-radius styling
- Responsive card layout
- CSS custom properties for consistent theming

## Technologies Used
- HTML5
- CSS3

## What I Learned
- How to use CSS custom properties (variables) for theme management
- How to create smooth transitions and hover animations
- How to style interactive elements with pseudo-classes (`:hover`, `:active`, `:focus`, `:visited`)
- How to use CSS `:target` selector for section visibility toggling
- How to apply decorative pseudo-elements (`::before`, `::after`)
- How to build dark-themed interfaces
- How to create accessible focus states with outlines

## Sections Included
- Greeting Card with Birthday Message
- Send Card Action Section
- Share on Social Media Section

## CSS Concepts Practiced
- CSS custom properties (variables)
- Pseudo-classes (`:hover`, `:active`, `:focus`, `:visited`, `:target`)
- Pseudo-elements (`::before`, `::after`)
- Transitions and animations
- Transform properties (scale, skewX)
- Box shadows and border-radius
- Flexbox layout
- Dark UI color schemes
- Text decoration and styling
