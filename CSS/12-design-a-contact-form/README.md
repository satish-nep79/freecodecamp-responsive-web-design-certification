# Design a Contact Form

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to design and style an engaging contact form using modern CSS techniques including gradients, glassmorphism, and accessible form controls.

## Features
- Full-page gradient background with soft radial accents
- Glassmorphic form container with `backdrop-filter` blur and translucent borders
- Clearly grouped form inputs with required-field indicators
- Accessible focus states with glow and border highlights
- Responsive input sizing and centered layout
- Decorative semi-circle and ring accents implemented with radial gradients

## Technologies Used
- HTML5
- CSS3

## What I Learned
- Building glassmorphism-style UI with translucent panels and blur
- Using CSS custom properties to manage theme colors and effects
- Creating accessible input focus and hover interactions
- Positioning decorative elements behind the content using absolute positioning and z-index

## Sections Included
- Name, Email, Message inputs
- Required-field indicators and placeholder styling
- Submit button with hover/press affordances

## CSS Concepts Practiced
- Linear and radial gradients
- `backdrop-filter` and translucent backgrounds
- Focus and hover visual feedback
- Absolute positioning and layering for decorative shapes
- Responsive container sizing and input constraints
