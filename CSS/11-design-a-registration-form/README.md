# Design a Registration Form

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to build and style a registration form focusing on clear field grouping, accessible inputs, and a dark-themed UI suitable for night-mode interfaces.

## Features
- Multi-field registration form with fieldset grouping and legends
- Required inputs with HTML validation (email, password pattern, required checkboxes)
- Custom inline controls for radio and checkbox inputs
- File upload field and age selector with min/max constraints
- Responsive form width with readable spacing and input sizing

## Technologies Used
- HTML5
- CSS3

## What I Learned
- How to structure forms semantically using `fieldset` and `legend`
- Designing accessible inline controls for radios and checkboxes
- Styling file inputs and responsive form containers
- Using dark background palettes with high-contrast inputs for readability

## Sections Included
- Personal details (first name, last name, email, password)
- Account type selection (personal, business)
- Profile picture upload, age, and referral select
- Bio textarea and terms acceptance checkbox
- Submit action

## CSS Concepts Practiced
- Responsive container sizing and max-width management
- Input and textarea styling for dark themes
- Fieldset and label layout for clear form sections
- Focus and hover affordances for interactive elements
