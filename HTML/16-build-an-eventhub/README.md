# Build an Event Hub

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create a structured event hub webpage using semantic HTML elements, featuring upcoming and past events with navigation links.

## Features
- Semantic HTML5 layout using `header`, `main`, `section`, and `article`
- Navigation menu with anchor links to page sections
- Upcoming events section with event details and dates
- Past events section with images and descriptions
- Organized event content using semantic structure

## Technologies Used
- HTML5

## What I Learned
- How to structure a multi-section webpage using semantic HTML
- How to use internal navigation with anchor links
- How to organize content using `article` elements
- How to properly include images with alt text
- How to build clear, accessible event-based layouts
