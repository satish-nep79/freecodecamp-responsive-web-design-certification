# Build a Confidential Email Page

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create a confidential email layout using HTML and CSS, focusing on visual hierarchy, overlay effects, typographic styling, and atmospheric presentation.

## Features
- Confidential email layout with a document-style container
- Background image with blur effect for visual depth
- Bold label styling for "CONFIDENTIAL" and "TOP SECRET"
- Blurred sensitive text that reveals on hover
- Responsive centered content with card-style formatting

## Technologies Used
- HTML5
- CSS3 (background images, pseudo-elements, transforms, hover states)

## What I Learned
- Using pseudo-elements to build blurred background layers
- Styling text and labels with strong visual contrast
- Creating hover reveal effects for sensitive content
- Positioning absolute labels inside a card layout
- Using box-shadow and border styling for document presentation
- Working with accessible typography and spacing for readability

## Sections Included
- Main confidential email container
- Prominent confidentiality alert labels
- Stylized recipient/sender information block
- Paragraph content with blurred sensitive phrases

## CSS Concepts Practiced
- Pseudo-elements (`::before`)
- Fixed background images and blur filters
- Absolute positioning and rotation transforms
- Box model with padding, margin, and border-radius
- Typographic emphasis with strong tags and semantic markup
- Hover transitions for interactive text reveals
