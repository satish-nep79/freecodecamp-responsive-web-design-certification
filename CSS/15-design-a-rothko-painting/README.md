# Design a Rothko-Inspired Painting

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this exercise was to recreate the aesthetic of a Rothko painting using HTML and CSS, focusing on color fields, soft edges, and vertical composition while practicing responsive layout and visual hierarchy.

## Features
- Stacked color fields that emulate Rothko's canvases
- Subtle gradients and box-shadows to create soft edges
- Responsive container that preserves aspect ratio on different screens
- Dark and light theme variations using CSS custom properties
- Accessible structure with semantic markup and high-contrast text option

## Technologies Used
- HTML5
- CSS3 (Custom Properties, Gradients, Flexbox)

## What I Learned
- Working with large color fields and contrast to create mood
- Using CSS variables to manage theme palettes
- Implementing aspect-ratio-friendly layouts with modern CSS
- Creating subtle depth with layered shadows and gradients

## Sections Included
- Canvas wrapper with responsive sizing
- Three stacked color blocks with independent styling
- Theme toggle sample (CSS-only) and high-contrast text example

## CSS Concepts Practiced
- CSS Custom Properties (variables)
- Radial and linear gradients
- Box-shadow layering for soft transitions
- Flexbox and alignment for vertical stacking
- Responsive units and container sizing
