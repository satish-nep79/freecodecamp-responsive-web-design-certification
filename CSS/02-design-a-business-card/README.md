# Design a Business Card

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to create and style a personal business card webpage using HTML and CSS, focusing on layout design, typography, spacing, and responsive styling.

## Features
- Styled business card layout
- Profile image with responsive sizing
- Personal information and contact details
- Portfolio link and social media links
- Centered card design with box shadow styling
- Custom background color and typography
- Clean and minimal user interface

## Technologies Used
- HTML5
- CSS3

## What I Learned
- How to structure profile card layouts using HTML
- How to style components using CSS
- How to center content using auto margins
- How to apply shadows and spacing for better UI design
- How to create responsive image layouts
- How to style links and typography consistently

## Sections Included
- Profile Information
- Contact Information
- Portfolio Link
- Social Media Links

## CSS Concepts Practiced
- Layout styling
- Typography
- Box shadows
- Responsive images
- Margin and padding
- Link styling
- Background colors
