# Design a Feature Selection Page

This project is part of the freeCodeCamp Responsive Web Design certification.

The goal of this project was to design and build an interactive feature selection interface with decorative visual elements, dark theme styling, and dynamic feature cards rendered with JavaScript.

## Features
- Dark-themed feature selection interface
- Decorative semi-circle radial gradients as background accents
- Interactive checkbox-based feature cards
- Dynamic card rendering using JavaScript template literals
- Smooth hover and active states with transitions
- Custom CSS variables for theming and consistency
- Accessible form controls with semantic labels
- Responsive card layout and typography

## Technologies Used
- HTML5
- CSS3 (Custom Properties, Gradients, Transitions)
- JavaScript (DOM manipulation, Template literals)

## What I Learned
- Creating complex gradient patterns with radial gradients
- Using CSS custom properties (variables) for theme management
- Building interactive UI components with proper state styling
- Dynamically rendering content with JavaScript
- Implementing smooth transitions and hover effects
- Dark mode design principles and color contrast
- Working with checkboxes in card-based layouts
- Positioning and layering with z-index

## Sections Included
- Main feature selection title and description
- Feature card container with dynamically rendered cards
- Four feature options: Analytics, AI Agent, Real-time Sync, and Collaboration
- Decorative background elements with multiple semi-circles

## CSS Concepts Practiced
- CSS Custom Properties (variables)
- Radial gradients for decorative effects
- Transitions and ease-in-out timing
- Checkbox styling and custom appearance
- Label styling for better UX
- Box shadows and layering
- Responsive typography and spacing
- Pseudo-classes (`:hover`, `:checked`, `:focus`)

## JavaScript Concepts Practiced
- DOM element selection with `querySelector`
- Template literal strings for dynamic HTML
- Array methods (`map`, `join`)
- innerHTML manipulation
- Data-driven UI rendering
